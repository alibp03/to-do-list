# To-Do List Application

این پروژه یک برنامه مدیریت تسک ساده است که با استفاده از React و TypeScript توسعه داده شده.

## معماری پروژه

این پروژه بر اساس معماری کامپوننتی React طراحی شده است. state
اصلی در TasksProvider نگه‌داری می‌شود و از طریق Context بین کامپوننت‌ها
به اشتراک گذاشته شده است. داده‌ها از طریق اکشن‌های مشخص وارد reducer
می‌شوند و UI بر اساس state به‌روزرسانی و رندر می‌شود. هر بخش از رابط
کاربری در قالب یک کامپوننت مستقل پیاده‌سازی شده تا قابلیت نگهداری،
خوانایی و توسعه‌پذیری پروژه افزایش یابد.

## دلیل انتخاب روش مدیریت state

در این پروژه از ترکیب React Context و useReducer استفاده شده‌است چون:
برای پروژه‌های کوچک تا متوسط، Redux یا Zustand نیاز نیست.
Context اجازه می‌دهد state به سادگی بین کامپوننت‌ها به اشتراک گذاشته شود.
